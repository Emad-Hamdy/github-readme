## Github projects 

A React-Redux app to display users projects

Technologies used:

* React
* Redux
* React Router
* CSS
* Webpack
* Github API


## Install
Clone the project and hit
``` 
npm install
```
to install the dependencies


## Run
Runs locally on port 8080
``` 
npm start 
```


## Build
Creates a bundle in build directory

``` 
npm run build
```
